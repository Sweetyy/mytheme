This is a note for 'mytheme' for Wordpress.

To use this theme make sure to install the plugin "Posts in page" https://github.com/ivycat/Posts-in-Page.

In the dashboard to set the menus please make sure to do the following instructions :
 - For the social menu : Go to 'Appearance' -> 'Menus' -> Create a new menu called 'menu_social' -> Add Custom link that will refer to
   the socials links -> check the checkbox 'Social Menu' in Theme locations at the bottom in the Menu Settings -> Save.
   Here are the icons for all the social links available that you have to put in the corresponding custom links of the social menu :
	- Twitter : <i class="fa fa-twitter-square" aria-hidden="true"></i>
	- Facebook : <i class="fa fa-facebook-official" aria-hidden="true"></i>
	- Instagram : <i class="fa fa-instagram" aria-hidden="true"></i>
	- Linkedin : <i class="fa fa-linkedin-square" aria-hidden="true"></i>
 - For the Main Menu : Same place -> Select the menu called 'main_nav' -> check the checkbox 'Main Menu' in Theme locations -> Save.
